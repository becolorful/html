AVERAGE =average(범위1, [범위2, ...]) 지정한 범위나 숫자의 산술 평균을 계산합니다.

=AVERAGE(A1:A5)는 A1부터 A5까지의 셀 값들의 평균을 반환합니다.

AVERAGEA =averagea(값1, [값2, ...]) 선택된 값들의 산술 평균을 계산합니다 (논리값, 텍스트 형식의 숫자 포함).

=AVERAGEA(A1, B1, C1)는 A1, B1, C1의 값들의 평균을 반환합니다.

AVERAGEIF =averageif(조건범위, 조건, [평균범위]) 조건을 만족하는 셀의 산술 평균을 계산합니다.

=AVERAGEIF(A1:A5, ">50")는 A1부터 A5까지의 셀 중에서 50보다 큰 값들의 평균을 반환합니다.

AVERAGEIFS =averageifs(평균범위, 조건범위1, 조건1, [조건범위2, 조건2, ...]) 범위 내 여러 개의 조건을 만족하는 값의 산술 평균을 계산합니다.

=AVERAGEIFS(A1:A5, B1:B5, ">50", C1:C5, "<100")는 B1부터 B5까지의 셀 값이 50보다 크고 C1부터 C5까지의 셀 값이 100보다 작은 값들의 평균을 반환합니다.

AVERAGE.WEIGHTED =average.weighted(값, 가중치, [추가 값, 추가 가중치]) 지정한 범위의 가중치를 고려한 산술 평균을 계산

=AVERAGE.WEIGHTED(A1:A5, B1:B5)는 A1부터 A5까지의 값에 대한 가중치를 B1부터 B5까지의 값으로 적용하여 가중 평균을 반환합니다.

COUNT =count(범위, [범위2, ...]) 해당 범위에서 숫자를 포함하고 있는 셀의 개수를 계산합니다.

=COUNT(A1:A5)는 A1부터 A5까지의 셀 중 숫자를 포함하는 셀의 개수를 반환합니다.

COUNTA =counta(범위, [범위2, ...]) 해당 범위에서 비어있지 않은 셀의 개수를 계산합니다.

=COUNTA(A1:A5)는 A1부터 A5까지의 셀 중 비어있지 않은 셀의 개수를 반환합니다.

COUNTBLANK =countblank(범위, [범위2, ...]) 해당 범위에서 비어있는 셀의 개수를 계산합니다.

=COUNTBLANK(A1:A5)는 A1부터 A5까지의 셀 중 비어있는 셀의 개수를 반환합니다.

COUNTUNIQUE =countunique(범위, [범위2, ...]) 해당 범위에서 고유 값의 개수 계산

=COUNTUNIQUE(A1:A5)는 A1부터 A5까지의 셀 중 고유 값의 개수를 반환합니다.

COUNTIF =countif(조건범위, 조건) 선택된 범위에서 조건을 만족하는 셀의 개수를 계산합니다.

=COUNTIF(A1:A5, ">50")는 A1부터 A5까지의 셀 중에서 50보다 큰 값들을 만족하는 셀의 개수를 반환합니다.

COUNTIFS =countifs(조건범위1, 조건1, [조건범위2, 조건2,...]) 선택된 범위에서 여러 개의 조건을 만족하는 셀의 개수를 계산합니다.

=COUNTIFS(A1:A5, ">50", B1:B5, "<100")는 A1부터 A5까지의 셀 중에서 50보다 크고, B1부터 B5까지의 셀 중에서 100보다 작은 값을 만족하는 셀의 개수를 반환합니다.

FREQUENCY =frequency(집합범위, 간격범위) 범위 내에서 값이 발생하는 빈도를 계산합니다.

=FREQUENCY(A1:A10, B1:B5)는 A1부터 A10까지의 값이 B1부터 B5까지의 간격에 해당하는 빈도를 계산합니다.

MODE =mode(범위, [범위2, ...]) 범위 내에서 가장 자주 발생한 값을 반환합니다.

=MODE(A1:A5)는 A1부터 A5까지의 값 중에서 가장 자주 발생한 값을 반환합니다.

MAX =max(범위, [범위2, ...]) 범위 내에서 가장 큰 값을 반환

=MAX(A1:A5)는 범위 A1부터 A5까지의 값 중 가장 큰 값을 반환합니다.
=MAX(A1:C3)는 범위 A1부터 C3까지의 값 중 가장 큰 값을 반환합니다.

MAXIFS =maxifs(최대값범위, 조건범위1, 조건1, [조건범위2, 조건2, ...]) 범위 내에서 조건들을 만족하는 가장 큰 값을 반환

=MAXIFS(A1:A5, B1:B5, ">10")는 범위 B1부터 B5까지의 값이 10보다 큰 경우에 대해 범위 A1부터 A5까지의 값 중 가장 큰 값을 반환합니다.
=MAXIFS(A1:C3, B1:B3, "<0", C1:C3, ">5")는 범위 B1부터 B3까지의 값이 0보다 작고, 범위 C1부터 C3까지의 값이 5보다 큰 경우에 대해 범위 A1부터 C3까지의 값 중 가장 큰 값을 반환합니다.

MIN =min(범위) 범위 내에서 가장 작은 값을 반환

=MIN(A1:A5)는 범위 A1부터 A5까지의 값 중 가장 작은 값을 반환합니다.
=MIN(A1:C3)는 범위 A1부터 C3까지의 값 중 가장 작은 값을 반환합니다.

MINIFS =minifs(최소값범위, 조건범위1, 조건1, [조건범위2, 조건2, ...]) 범위 내에서 조건들을 만족하는 가장 작은 값을 반환

=MINIFS(A1:A5, B1:B5, ">10")는 범위 B1부터 B5까지의 값이 10보다 큰 경우에 대해 범위 A1부터 A5까지의 값 중 가장 작은 값을 반환합니다.
=MINIFS(A1:C3, B1:B3, "<0", C1:C3, ">5")는 범위 B1부터 B3까지의 값이 0보다 작고, 범위 C1부터 C3까지의 값이 5보다 큰 경우에 대해 범위 A1부터 C3까지의 값 중 가장 작은 값을 반환합니다.

LARGE =large(범위, k) 범위 내에서 k번째로 큰 값을 반환

=LARGE(A1:A5, 2)는 범위 A1부터 A5까지의 값 중 두 번째로 큰 값을 반환합니다.
=LARGE(A1:C3, 3)는 범위 A1부터 C3까지의 값 중 세 번째로 큰 값을 반환합니다.

SMALL =small(범위, k) 범위 내에서 k번째로 작은 값을 반환

=SMALL(A1:A5, 2)는 범위 A1부터 A5까지의 값 중 두 번째로 작은 값을 반환합니다.
=SMALL(A1:C3, 3)는 범위 A1부터 C3까지의 값 중 세 번째로 작은 값을 반환합니다.

MEDIAN =median(범위, [범위2, ...]) 값들 중 중간 값을 반환

=MEDIAN(A1:A5)는 범위 A1부터 A5까지의 값들 중 중간 값을 반환합니다.
=MEDIAN(A1:C3)는 범위 A1부터 C3까지의 값들 중 중간 값을 반환합니다.

RANK =rank(값, 범위, [정렬방향]) 특정 값의 범위 내 순위를 반환

=RANK(A1, A1:A5)는 값 A1이 범위 A1부터 A5까지에서 몇 번째로 큰 값인지를 반환합니다.
=RANK(A1, A1:C3, 1)는 값 A1이 범위 A1부터 C3까지에서 내림차순으로 정렬되었을 때 몇 번째로 큰 값인지를 반환합니다.

RANK.AVG =rank.avg(값, 범위, [정렬방향]) 범위 내 크기 순위를 계산한 뒤 해당 값의 평균 순위를 반환

=RANK.AVG(A1, A1:A5)는 값 A1이 범위 A1부터 A5까지에서 크기 순위를 계산한 뒤 해당 값의 평균 순위를 반환합니다.
=RANK.AVG(A1, A1:C3, 0)는 값 A1이 범위 A1부터 C3까지에서 오름차순으로 정렬되었을 때 크기 순위를 계산한 뒤 해당 값의 평균 순위를 반환합니다.
