IF =if(조건, 참값, 거짓값) 조건을 검사하여 참인 경우 참값을, 거짓인 경우 거짓값을 반환

=IF(A1>5, "큼", "작음")
셀 A1의 값이 5보다 큰 경우 "큼"을 반환, 그렇지 않은 경우 "작음"을 반환합니다.

AND =and(조건1, 조건2, ...) 모든 조건이 참인 경우 참을 반환

=AND(A1>5, A2<10)
셀 A1의 값이 5보다 크고, A2의 값이 10보다 작은 경우 참을 반환합니다.

OR =or(조건1, 조건2, ...) 하나 이상의 조건이 참인 경우 참을 반환

=OR(A1>5, A2<10)
셀 A1의 값이 5보다 크거나, A2의 값이 10보다 작은 경우 참을 반환합니다.

NOT =not(조건) 조건의 반대를 반환

=NOT(A1>5)
셀 A1의 값이 5보다 크지 않은 경우 참을 반환합니다.

LEFT =left(문자열, k) 문자열의 왼쪽에서부터 k번째까지의 문자 추출

=LEFT("Hello World", 5) 결과는 "Hello"가 됩니다.

RIGHT =right(문자열, k) 문자열의 오른쪽에서부터 k번째까지의 문자 추출

=RIGHT("Hello World", 5) 결과는 "World"가 됩니다.

MID =mid(문자열, k, n) 문자열의 왼쪽에서 k번째 문자부터 n개의 문자 추출

=MID("Hello World", 7, 5) 결과는 "World"가 됩니다.

TEXTJOIN =textjoin(구분 기호, 빈칸무시, 범위, [범위2, 범위3, ...]) 여러 문자열을 구분 기호로 나눠 하나의 문장으로 결합

=TEXTJOIN(", ", TRUE, A1:A3) 결과는 "Apple, Banana, Orange"가 됩니다.

SPLIT =split(문자열, 구분자, [문자 분할, 빈 문자 삭제]) 구분자를 사용해 문자열을 분할

=SPLIT("Apple, Banana, Orange", ", ") 결과는 {"Apple"; "Banana"; "Orange"}가 됩니다.

TEXTSPLIT =textsplit(문자열, 열 구분자, [행 구분자, 연속 구분 기호 무시, 대/소문자 구분]) 열과 행 구분자를 사용해 문자열을 분할

=TEXTSPLIT("Apple,Banana;Orange", ",", ";") 결과는 {"Apple"; "Banana"; "Orange"}가 됩니다.

LOWER =lower(문자열) 지정된 문자열을 소문자로 변환

=LOWER("Hello World") 결과는 "hello world"가 됩니다.

UPPER =upper(문자열) 지정된 문자열을 대문자로 변환

=UPPER("Hello World") 결과는 "HELLO WORLD"가 됩니다.

PROPER =proper(문자열) 각 단어의 첫번째 글자를 대문자로 변환

=PROPER("hello world") 결과는 "Hello World"가 됩니다.

REPLACE =replace(문자열, 시작지점(1~), 길이, 새 문자열) 문자열의 일부분을 다른 문자열로 대체

=REPLACE("Hello World", 7, 5, "Universe") 결과는 "Hello Universe"가 됩니다.

SUBSTITUTE =substitute(문자열, 찾을 문자, 새 문자열, [찾을 바꿀 지점]) 문자에서 특정 문자를 찾아 다른 문자로 대체

=SUBSTITUTE("Hello World", "o", "i") 결과는 "Helli World"가 됩니다.

TEXT =text(값, 표시형식) 특정 문자열 또는 셀의 값을 원하는 서식의 문자열로 변환

=TEXT(TODAY(), "yyyy-mm-dd") 결과는 현재 날짜를 "2023-06-17"과 같은 형식으로 표시합니다.

PHONETIC =phonetic(문자열) 한자/일본어의 윗주(후리가나)를 추출/반환

=PHONETIC("漢字") 결과는 "かんじ"가 됩니다.

CHAR =char(숫자) 입력된 수를 유니코드표에 해당되는 문자로 반환

=CHAR(65) 결과는 "A"가 됩니다.

CODE =code(문자열) 입력된 문자열의 첫 번째 문자를 나타내는 유니코드 숫자 값을 반환

=CODE("A") 결과는 65가 됩니다.

VALUE =value(문자열) 숫자 서식으로 인식되는 문자열을 숫자로 변환

=VALUE("123") 결과는 123이 됩니다.

LEN =len(문자열) 선택된 셀 또는 문자열의 글자수를 반환

=LEN("Hello") 결과는 5가 됩니다.

SEARCH =search(찾을 문자, 문자열, [시작 지점]) 문자열에서 특정 문자가 처음으로 발견된 지점을 반환 (대소문자 구분)

=SEARCH("o", "Hello World") 결과는 5가 됩니다.

FIND =find(찾을 문자, 문자열, [시작 지점]) 문자열에서 특정 문자가 처음으로 발견된 지점을 반환 (대소문자 무시)

=FIND("o", "Hello World") 결과는 5가 됩니다.
